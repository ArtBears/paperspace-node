(static) create(params, cb) → {object}

Source:
  • snapshots/create.js, line 6

Create a snapshot of the machine that has the given id. This action is only available if your credentials have the privilege to access the machine from which the snapshot has been created (either the individual user who owns the machine, or the administrator of the team).

Examples
paperspace.snapshots.create({
  machineId: 123,
  name: 'My Snapshot 1',
}, function(err, resp) {
  // handle error or http response
});
$ paperspace snapshots create \
    --machineId 123 \
    --name "My Snapshot 1"
# HTTP request:
POST /machines/123/createSnapshot {"name": "My Snapshot 1"}
# Returns 201 on success
