(static) autoshutdown(params, cb)

Source:
  • machines/autoshutdown.js, line 6

Configure a machine to shut down automatically after a given interval.

Examples
paperspace.machines.autoshutdown({
  machineId: 123,
  interval: '1 week'
}, function(err, resp) {
  // handle error or http response
});
$ paperspace machines autoshutdown --machineId 123 --interval "1 week"

(static) create(params, cb)

Source:
  • machines/create.js, line 6

Create a new Paperspace virtual machine. If you are using an individual account, you will be assigned as the owner of the machine. If you are a team administrator, you must specify who the machine should belong to, either by their user id, or their email address. If you pass a full set of user parameters, we will create a user within your team and assign them the machine.

Examples
paperspace.machines.create({
  location: 'nyc01',
  machineType: 'Air',
  size: '50GB',
  billingType: 'hourly',
  machineName: 'My Machine 1',
}, function(err, resp) {
  // handle error or http response
});
$ paperspace machines create --location "nyc01" \
   --machineType "Air" --size "50GB" --billingType "hourly" \
   --machineName "My Machine 1"
