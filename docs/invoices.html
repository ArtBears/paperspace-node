(static) list(params, cb) → {object}

Source:
  • invoices/list.js, line 6

List the invoices for a given team account or user account. If a user id is passed, invoices for that user will be returned. If the user is a team administrator, invoices for the team will be returned. If a team id is passed, invoices for the team will be returned. This action is only permitted for credentials that have privileges for the account in question.

Examples
paperspace.invoices.list({
  userId: 123, // optional - Either userId or teamId, not both!
  teamId: 456, // optional - Either userId or teamId, not both!
}, function(err, resp) {
  // handle error or http response
});
$ paperspace invoices list --userId 123
# HTTP request:
GET /invoices/listInvoices {"userId": 123, "teamId": 456} # One or the other, but not both!
# Returns 200 on success

(static) show(params, cb) → {object}

Source:
  • invoices/show.js, line 6

Show an individual invoice, looking it up by its id. This action is only permitted for credentials have privileges for the account that receives the invoice. (In the case of a team, this is the team administrator.)

Examples
paperspace.invoices.show({
  invoiceId: 123,
}, function(err, resp) {
  // handle error or http response
});
$ paperspace invoices show --invoiceId 123
# HTTP request:
GET /invoices/123
# Returns 200 on success
