(static) addmember(params, cb)

Source:
  • teams/addmember.js, line 6

Add a new member (user account) to the team with the given id. When created, the account will immediately be available for login by the person with the credentials assigned. No machine is created with this action. This action can only be performed by the administrator of the team.

Examples
paperspace.teams.addmember({
  teamId: 123,
  email: 'example@example.com',
  password: 'secret123',
}, function(err, resp) {
  // handle error or http response
});
$ paperspace teams addmember \
    --teamId 123 \
    --email "example@example.com" \
    --password "secret123"
# HTTP request:
POST /teams/123/addMember {"email": "example@example.com", "password": "secret123"}
# Returns 201 on success

(static) removemember(params, cb)

Source:
  • teams/removemember.js, line 6

Remove a member from a team. This action only works if the user with the given user id belongs to the team with the given team id. This action may only be performed by the team's administrator. All of the user's machines are deactivated immediately when this action succeeds. Any snapshots of those machines are also destroyed. Access to the user account will also be terminated.

Examples
paperspace.teams.removemember({
  teamId: 123,
  userId: 456,
}, function(err, resp) {
  // handle error or http response
});
$ paperspace teams removemember \
    --teamId 123 \
    --userId 456
# HTTP request:
POST /teams/123/removeMember {"userId": 456}
# Returns 204 on success

Properties

Id of the user to remove from the team

(static) show(params, cb)

Source:
  • teams/show.js, line 6

Show information about an individual team using the given team id. This returns basic information about the team, as well as information about all members' user accounts, and the team administrator's account. This action is only permitted for the team's administrator.

Examples
paperspace.teams.show({
  teamId: 123,
}, function(err, resp) {
  // handle error or http response
});
$ paperspace teams show --teamId 123
# HTTP request:
GET /teams/123
# Returns 200 on success
